วิธีแปลงรูปหลายรูปเป็นไฟล์ PDF

การแปลงไฟล์รูปภาพหลาย ๆ ไฟล์ให้รวมเป็นไฟล์ PDF เดียวสามารถทำได้หลายวิธี ขึ้นอยู่กับอุปกรณ์ที่คุณใช้งาน ไม่ว่าจะเป็น Windows, macOS, iPhone/iPad, Android หรือผ่านเว็บเบราว์เซอร์ หน้านี้จะอธิบายทุกวิธีในหน้าเดียวเพื่อให้ง่ายต่อการเข้าใจ

วิธีบน Windows 10/11

  1. รวมรูปทั้งหมดไว้ในโฟลเดอร์เดียวกัน และเรียงชื่อไฟล์ตามลำดับที่ต้องการ (เช่น 01, 02, 03...)
  2. เลือกไฟล์ทั้งหมด → คลิกขวา → Print
  3. ใน Printer ให้เลือก Microsoft Print to PDF
  4. ตั้งค่า Paper size, Quality และเลือกว่าจะ Fit picture to frame หรือไม่
  5. กด Print แล้วบันทึกเป็นไฟล์ PDF

วิธีบน macOS (Preview)

  1. เลือกไฟล์รูปทั้งหมด → คลิกขวา → Open With → Preview
  2. ในแถบข้างซ้าย คุณสามารถลากเพื่อจัดลำดับรูปได้
  3. ไปที่เมนู File → Print… → มุมซ้ายล่างเลือก Save as PDF
  4. ตั้งชื่อและเลือกตำแหน่งไฟล์ PDF

วิธีบน iPhone / iPad

  1. เปิดแอป Photos → เลือกรูปทั้งหมด → กดปุ่มแชร์ → เลือก Print
  2. ที่หน้าพรีวิวพิมพ์ ใช้นิ้ว ถ่าง (pinch out) เพื่อแปลงเป็น PDF
  3. กดปุ่มแชร์อีกครั้ง → บันทึกไปที่ Files หรือส่งต่อไปยังแอปอื่น

วิธีบน Android

  1. เปิด Google Photos หรือแอปดูรูป → เลือกหลายรูป
  2. เลือก Print หรือ Share → เลือกเครื่องพิมพ์เป็น Save as PDF
  3. ตั้งค่าหน้ากระดาษ → บันทึกเป็นไฟล์ PDF

วิธีแปลงผ่านเว็บ

  1. เข้าเว็บไซต์แปลงไฟล์ที่น่าเชื่อถือ เช่น iLovePDF, Smallpdf
  2. อัปโหลดไฟล์รูปทั้งหมด
  3. จัดเรียงลำดับรูป
  4. กด Convert → ดาวน์โหลดไฟล์ PDF
หมายเหตุ: หากเป็นเอกสารสำคัญหรือข้อมูลส่วนตัว ควรใช้วิธีแปลงบนเครื่อง (Windows/macOS) แทนการอัปโหลดไปยังเว็บไซต์ เพื่อความปลอดภัยของข้อมูล

ทิปเพิ่มเติม

  • ตั้งชื่อไฟล์รูปด้วยตัวเลขนำหน้าเพื่อควบคุมลำดับใน PDF
  • ถ้าต้องการไฟล์เล็กลง ให้เลือกความละเอียด (dpi) ต่ำ เช่น 150–200 dpi
  • สำหรับภาพถ่าย ควรใช้ JPG แทน PNG เพื่อให้ไฟล์ PDF มีขนาดเล็กลง
